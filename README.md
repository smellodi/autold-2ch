## Development environment and features

- Microsoft Visual Studio Community 2019, Version 16.9.4 (May 2021)
- .NET Core 3.1 (LTS)
- C# 9.0
- Tested only as x86-32 application.

## Dependencies

NuGet packages:
- System.IO.Ports
- System.Management
- NAudio
- WPFLocalizeExtension
- ScottPlot.WPF
